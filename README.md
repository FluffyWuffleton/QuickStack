# QuickStack
Easily stack inventory contents to/from locations with matching items.

# Default keybinds
## Deposit
*Shift+/* -- main inv. -> nearby
*Shift+Alt+/* -- full inv. -> nearby
## Collect
*Ctrl+/* -- nearby -> main inv.
*Ctrl+Alt+/* -- nearby -> full inv.
